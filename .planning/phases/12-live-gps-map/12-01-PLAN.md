2. Create `src/lib/maps/vehicle-status.ts`:
   - Export type `VehicleStatus = 'moving' | 'idle' | 'offline'`
   - Export function `getVehicleStatus(speed: number | null, lastUpdate: Date): VehicleStatus`
     - Offline: no GPS update in 10+ minutes (Date.now() - lastUpdate.getTime() > 10 * 60 * 1000)
     - Idle: speed is null or < 5 mph
     - Moving: speed >= 5 mph
   - Export const `STATUS_COLORS` mapping: `{ moving: { bg: 'bg-green-500', border: 'border-green-700' }, idle: { bg: 'bg-yellow-500', border: 'border-yellow-700' }, offline: { bg: 'bg-red-500', border: 'border-red-700' } }`

3. Create `src/lib/maps/map-utils.ts`:
   - Export interface `VehicleLocation` with fields: id (string), truckId (string), latitude (number), longitude (number), speed (number | null), heading (number | null), timestamp (Date), truck (object with make, model, licensePlate strings)
   - Export function `calculateBounds(vehicles: VehicleLocation[]): [[number, number], [number, number]] | null` that uses `@turf/turf` bbox() to calculate map bounds from vehicle positions. Return null if vehicles array is empty. Convert turf bbox [minX, minY, maxX, maxY] to Leaflet bounds format [[minY, minX], [maxY, maxX]] (note: turf uses [lng, lat], Leaflet uses [lat, lng]).
   - Export const `DEFAULT_CENTER: [number, number] = [39.8283, -98.5795]` (US center)
   - Export const `DEFAULT_ZOOM = 5`

4. Create `src/app/(owner)/live-map/actions.ts`:
   - Add `'use server'` directive
   - Import `requireRole` from `@/lib/auth/server`, `UserRole` from `@/lib/auth/roles`, `getTenantPrisma` from `@/lib/context/tenant-context`
   - Export async function `getLatestVehicleLocations()`:
     - Call `await requireRole([UserRole.OWNER, UserRole.MANAGER])`
     - Get `db` via `await getTenantPrisma()`
     - Use raw SQL query with `DISTINCT ON` to get the most recent GPS location per truck (same pattern as research example):
       ```sql
       SELECT DISTINCT ON (gps."truckId")
         gps.id, gps."truckId", gps.latitude, gps.longitude, gps.speed, gps.heading, gps.timestamp,
         t.make, t.model, t."licensePlate"
       FROM "GPSLocation" gps
       INNER JOIN "Truck" t ON gps."truckId" = t.id
       WHERE gps."tenantId" = ${tenantId}::uuid
       ORDER BY gps."truckId", gps.timestamp DESC
       ```
     - NOTE: To get tenantId for the raw query, use `requireTenantId()` from `@/lib/context/tenant-context` (not getTenantPrisma). Import it separately.
     - Map results to convert Decimal lat/lng to Number (critical: Prisma Decimal -> JavaScript number).
     - Return array of VehicleLocation objects (matching the interface from map-utils.ts).
     - Use `// @ts-ignore` for raw query typing per project pattern.
  </action>
