Add two new server actions to the EXISTING `src/app/(owner)/live-map/actions.ts` file (append, do NOT replace existing getLatestVehicleLocations):

1. Export async function `getVehicleRouteHistory(truckId: string, hoursBack: number = 24)`:
   - Call `await requireRole([UserRole.OWNER, UserRole.MANAGER])`
   - Get `db` via `await getTenantPrisma()`
   - Calculate cutoff: `new Date(Date.now() - hoursBack * 60 * 60 * 1000)`
   - Query GPSLocation with `findMany`:
     - where: `{ truckId, timestamp: { gte: cutoffTime } }` (RLS handles tenantId automatically via getTenantPrisma)
     - orderBy: `{ timestamp: 'asc' }`
     - select: `{ latitude: true, longitude: true, speed: true, timestamp: true }`
   - Use `// @ts-ignore` for Prisma 7 extension type issue
   - Map results: convert Decimal lat/lng to Number (`Number(loc.latitude)`, `Number(loc.longitude)`)
   - Return array of `{ latitude: number, longitude: number, speed: number | null, timestamp: Date }`

2. Export async function `getVehicleDiagnostics(truckId: string)`:
   - Call `await requireRole([UserRole.OWNER, UserRole.MANAGER])`
   - Get `db` via `await getTenantPrisma()`
   - Three parallel queries (use Promise.all):
     a. Truck data: `db.truck.findUnique({ where: { id: truckId }, select: { make, model, year, licensePlate, odometer } })`
     b. Latest GPS: `db.gPSLocation.findFirst({ where: { truckId }, orderBy: { timestamp: 'desc' }, select: { latitude, longitude, speed, heading, timestamp } })`
     c. Latest fuel: `db.fuelRecord.findFirst({ where: { truckId }, orderBy: { timestamp: 'desc' }, select: { quantity, timestamp, odometer } })`
   - Use `// @ts-ignore` before each Prisma query
   - If truck not found, throw `new Error('Truck not found')`
   - Derive engine state from speed: `speed > 5 ? 'running' : speed > 0 ? 'idle' : 'off'`. If no latestGPS, engine state is `'off'`.
   - Estimate fuel level (simplified heuristic per research):
     - Assume 6 MPG and 150 gallon tank
     - `milesSinceLastFill = truck.odometer - Number(latestFuel.odometer)` (if latestFuel exists)
     - `estimatedGallonsUsed = milesSinceLastFill / 6`
     - `estimatedGallonsRemaining = Math.max(0, Number(latestFuel.quantity) - estimatedGallonsUsed)`
     - `estimatedFuelLevel = Math.round((estimatedGallonsRemaining / 150) * 100)`
     - If no fuel record, default to 50%
     - Clamp to 0-100 range
   - Convert Decimal fields to Number in the return value
   - Return object with: truck, latestGPS (with Number-converted coords or null), latestFuel (with Number-converted quantity or null), engineState, estimatedFuelLevel
