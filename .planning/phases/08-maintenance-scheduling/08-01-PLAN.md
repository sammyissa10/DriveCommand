**Create `src/lib/utils/maintenance-utils.ts`:**

Export `calculateNextDue(schedule, currentOdometer)` function:
- Input: schedule object with { intervalDays, intervalMiles, baselineDate, baselineOdometer } (all nullable intervals), currentOdometer: number
- Output: DueStatus object with { isDueByDate, isDueByMileage, isDue, nextDueDate, nextDueMileage, daysUntilDue, milesUntilDue }
- Date calculation: Use `new Date(baselineDate)` then `.setDate(getDate() + intervalDays)` (NOT millisecond math — avoids DST bugs per research)
- isDue = isDueByDate || isDueByMileage (whichever comes first)
- Handle null intervals gracefully (if intervalDays is null, date-based fields are null)
- Export the DueStatus interface for use in UI components
