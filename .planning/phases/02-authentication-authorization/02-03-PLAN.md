   - Export `RoleGuard` component with props: `children: React.ReactNode`, `allowedRoles: UserRole[]`, `fallback?: React.ReactNode`.
   - Implementation:
     - Call `useUser()` to get `user` and `isLoaded`.
     - If `!isLoaded`, return `null` (prevent flash of content while loading).
     - Read role from `user?.publicMetadata?.role` and cast as `UserRole | undefined`.
     - If role is not in allowedRoles, render `fallback` (defaults to `null`).
     - Otherwise, render `children`.
   - This is for UI VISIBILITY only, NOT security. All security enforced server-side.
