     - Type select: PICKUP / DELIVERY dropdown
     - Address: Use `AddressAutocomplete` component (import already exists in the file). The `name` prop should NOT be set on AddressAutocomplete since we use hidden fields. Instead, use `onPlaceSelect` to update the stop's address in state.

7. The hidden fields for each stop use the current array index as N (NOT the clientId). This ensures the server action receives `stops_0_address`, `stops_1_address`, etc. in sequential order matching the displayed order.
