**Action 2: `getFuelEfficiencyTrend(daysBack = 30)`** (FUEL-01)
- Role check same as above
- Raw SQL: Daily fuel efficiency by aggregating fill-ups per day
  ```sql
  SELECT
    DATE(timestamp) as date,
    SUM(quantity)::float as gallons,
    SUM("totalCost")::float as cost,
    MAX(odometer) - MIN(odometer) as miles
  FROM "FuelRecord"
  WHERE "tenantId" = $1::uuid AND timestamp >= $2
  GROUP BY DATE(timestamp)
  ORDER BY date ASC
  ```
- Initialize all dates in range (like safety trend pattern) with zeros
- For each day with data: compute MPG = miles/gallons (use calculateMPG)
- Return: Array of `{ date: string, mpg: number, gallons: number, cost: number }`

**Action 3: `getCO2Emissions(daysBack = 30)`** (FUEL-02)
- Role check same
- Raw SQL: Per-truck fuel consumption for CO2 calculation
  ```sql
  SELECT
    t.id as "truckId",
    t.make,
    t.model,
    t."licensePlate",
    SUM(fr.quantity)::float as "totalGallons",
    SUM(fr."totalCost")::float as "totalCost"
  FROM "Truck" t
  LEFT JOIN "FuelRecord" fr ON t.id = fr."truckId" AND fr.timestamp >= $2 AND fr."tenantId" = $1::uuid
  WHERE t."tenantId" = $1::uuid
  GROUP BY t.id, t.make, t.model, t."licensePlate"
  ORDER BY "totalGallons" DESC NULLS LAST
  ```
- For each truck: compute CO2 using `calculateCO2Emissions(totalGallons)`
- Compute fleet total CO2
- Return: `{ trucks: Array<{truckId, name, licensePlate, totalGallons, co2Kg}>, fleetTotalCO2, fleetTotalGallons, methodology: "EPA standard: 8.887 kg CO2 per gallon of diesel" }`

**Action 4: `getIdleTimeAnalysis(daysBack = 30)`** (FUEL-03)
- Role check same
- Raw SQL on GPSLocation: For each truck, count total GPS points and points where speed = 0 (or speed IS NULL with timestamp in range, meaning stationary/off)
  ```sql
  SELECT
    t.id as "truckId",
    t.make,
    t.model,
    t."licensePlate",
    COUNT(g.id)::int as "totalPoints",
    SUM(CASE WHEN g.speed IS NOT NULL AND g.speed = 0 THEN 1 ELSE 0 END)::int as "idlePoints",
    SUM(CASE WHEN g.speed IS NOT NULL AND g.speed > 0 THEN 1 ELSE 0 END)::int as "movingPoints"
  FROM "Truck" t
  LEFT JOIN "GPSLocation" g ON t.id = g."truckId" AND g.timestamp >= $2 AND g."tenantId" = $1::uuid
  WHERE t."tenantId" = $1::uuid
  GROUP BY t.id, t.make, t.model, t."licensePlate"
  ```
- For each truck: calculate idlePercent = (idlePoints / totalPoints * 100) or 0 if no points
- Estimate idle cost: (idlePoints * avgMinutesBetweenReadings / 60) * IDLE_COST_PER_HOUR. Use 5 minutes as average interval between GPS readings (seed data generates points every ~5 min equivalent).
- Return: Array of `{ truckId, name, licensePlate, totalPoints, idlePoints, movingPoints, idlePercent, estimatedIdleCost }`

**Action 5: `getFuelEfficiencyRankings(daysBack = 30)`** (FUEL-04)
- Role check same
- Raw SQL: Per-truck MPG ranking using LEFT JOIN (same pattern as safety getDriverRankings)
  ```sql
  SELECT
    t.id as "truckId",
    t.make,
    t.model,
    t."licensePlate",
    COALESCE(SUM(fr.quantity), 0)::float as "totalGallons",
    COALESCE(SUM(fr."totalCost"), 0)::float as "totalCost",
    COALESCE(MAX(fr.odometer) - MIN(fr.odometer), 0)::int as "totalMiles",
    COUNT(fr.id)::int as "fillUpCount"
  FROM "Truck" t
  LEFT JOIN "FuelRecord" fr ON t.id = fr."truckId" AND fr.timestamp >= $2 AND fr."tenantId" = $1::uuid
  WHERE t."tenantId" = $1::uuid
  GROUP BY t.id, t.make, t.model, t."licensePlate"
  ORDER BY "totalMiles" DESC NULLS LAST
  ```
- For each truck: compute MPG using `calculateMPG(totalMiles, totalGallons)`, costPerMile using `calculateCostPerMile(totalCost, totalMiles)`
- Sort by MPG descending (best first), trucks with 0 fuel records at bottom
- Return: Array of `{ truckId, name, licensePlate, mpg, totalGallons, totalCost, totalMiles, costPerMile, fillUpCount }`
